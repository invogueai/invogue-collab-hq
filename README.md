# Invogue Collab HQ

Influencer marketing management system for Invogue shapewear brand.

## Features
- **Role-based access**: Admin, Manager, Negotiator, Finance, Logistics
- **Deal workflow**: Create → Approve → Email → Ship → Content → Invoice → Pay
- **Deliverables tracking**: Per-deliverable status across all influencers
- **Payment management**: Advance, partial, and final payments with running balance
- **Influencer database**: Centralized profiles with collab history
- **Campaign budgeting**: Budget tracking with utilization alerts
- **Logistics**: Shipment dispatch and tracking
- **Audit trail**: Complete activity log
